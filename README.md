# FileWall – Pre-launch Survey & Waitlist

This repository contains the pre-launch website for **FileWall**, designed to collect early user feedback, validate product direction, and onboard initial users.

---

## 🎯 Purpose

Before building at scale, this platform is used to:

- Gather user feedback through surveys
- Understand real user pain points
- Validate pricing and feature assumptions
- Collect early access signups (waitlist)

This helps ensure FileWall is built based on **real demand**, not assumptions.

---

## 🧠 What This Includes

- Landing page (product overview)
- Survey / feedback collection
- Waitlist / early access signup
- Blogs / Blogs about the product
